# Average_temp_Seoul-Prophet-

시계열 데이터를 다뤄보기 위해 데이콘에서 진행하는 서울시 평균 기온예측 해커톤에 참여하였습니다.

모델선정 : AutoML 패키지는 사용이 불가능하였고 Prophet모델은 Facebook에서 개발한 시계열 예측을 위한 오픈 소스 라이브러리이기 때문에 사용이 가능했다.
